This document is the API specification for the jscep SCEP library.

          Description

Packages
org.jscep.asn1 Provides classes and interfaces to represent ASN.1 objects specific to SCEP.
org.jscep.client Provides classes to represent a SCEP client.
org.jscep.client.verification Provides an interface and several implementation classes for verifying the identity of a certificate.
org.jscep.message Provides classes and interfaces to represent SCEP Secure Message Objects.
org.jscep.server Provides classes to represent a SCEP server.
org.jscep.transaction Provides classes and interfaces to manipulate SCEP transactions.
org.jscep.transport Provides classes for providing HTTP transport to SCEP messages.
org.jscep.transport.request Provides classes and interfaces to represent SCEP requests.
org.jscep.transport.response Provides classes and interfaces to handle and represent SCEP responses.
org.jscep.util Provides classes for performing miscellaneous functions.

 

This document is the API specification for the jscep SCEP library.

For more information about the jscep project, please visit the project web site at jscep.org, where you will find the latest documentation, source code and examples.
