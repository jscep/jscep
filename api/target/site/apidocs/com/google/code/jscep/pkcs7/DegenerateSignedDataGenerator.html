Class DegenerateSignedDataGenerator

java.lang.Object
  com.google.code.jscep.pkcs7.DegenerateSignedDataGenerator

public class DegenerateSignedDataGenerator
extends Object

This class is used for generating degenerate (certificates and CRLs only) SignedData instances.

Example usage: 

 X509Certificate cert = ...;
 
 DegenerateSignedDataGenerator gen = new DegenerateSignedDataGenerator();
 gen.addCertificate(cert);
 SignedData signedData = gen.generate();
